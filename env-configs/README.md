# 매체별 환경변수

Vercel에서 프로젝트 생성 후, 각 매체에 맞는 환경변수를 설정하세요.
같은 GitHub 레포(vs-media-template)를 9번 Import하고, 환경변수만 다르게 넣으면 됩니다.

## Vercel 설정 방법
1. https://vercel.com/new 접속
2. "Import Git Repository" → `tames-hub/vs-media-template` 선택
3. Project Name을 매체명으로 설정 (예: sparksquare)
4. Environment Variables에 아래 값 입력
5. Deploy
6. Settings → Domains에서 도메인 연결
7. 9개 매체 반복
